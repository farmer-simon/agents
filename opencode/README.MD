# openCode 配置说明
## 配置目录
### Window
```
	C:\Users\用户名\.config\opencode
```
### Mac
```
	~/.config/opencode/skills/
```

## 目录结构
```
# 技能定义
~/.config/opencode/skills/
├── dev-builder/
│   └── SKILL.md
└── product-spec-builder/
    └── SKILL.md

# 命令定义
~/.config/opencode/commands
└── ui.md
└── prd.md
```

## AGENTS 约定
AGENTS.md 放在项目根目录
